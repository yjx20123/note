###### 1、模拟添加设备

​	请求方式：`post`

​	请求路径：http://agric.lo3.cn/simulatation/adddevice

​	请求参数：num(生成条数)

> 示例:http://agric.lo3.cn/simulatation/adddevice?num=1

​	请求返回：

```json
{
  "status_code": 200,
  "detail": "添加设备成功",
  "headers": null
}
```

###### 3、获取设备状态数量

请求方式：`get`

​	请求路径：http://agric.lo3.cn/echart/num

​	请求参数：id

> 示例：http://agric.lo3.cn/echart/num?id=1

请求返回：

```json
{
  "code": 200,
  "message": "Success",
  "data": {
    "online": 2,
    "stanby": 0,
    "fault": 1,
    "offline": 0
  }
}
```

**online**:在线

__stanby__:待机

__fault__:损坏

**offline**:离线
