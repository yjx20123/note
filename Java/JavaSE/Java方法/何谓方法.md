#### Java方法是语句的集合，他们在一起执行一个功能

 	方法是解决一类问题的步骤的有序组合

方法包含于类或者对象中

方法在程序中创建，在其他地方被引用

#### 设计方法的原则：方法的本意是功能块，就是实现某个功能的语句块集合。我们设计方法的时候，最好保持方法的原子性，```就是一个方法只完成一个功能```，这样利于后期的扩展
